# -
致敬小时候最爱看的节目哈哈哈哈，python写了一个仿开门大吉流程的猜歌挑战系统(●'◡'●)
### 核心功能
- **用户系统**：注册/登录、密码加密存储、个人资料管理
- **游戏机制**：12扇门闯关、积分累计、求助功能
- **数据持久化**：用户数据本地文件存储
- **排行榜系统**：历史最高分排名

### 技术栈
- **前端**：Tkinter（Python标准GUI库）
- **后端**：Python 3.8+
- **数据存储**：自定义文件格式（类JSON结构）
- **音频处理**：playsound第三方库 + 多线程播放
- **安全机制**：MD5密码加密

### 项目亮点
完整的用户认证系统：实现注册、登录、密码加密、个人信息管理
模块化设计：采用MVC思想，业务逻辑与界面分离
多线程音频播放：避免播放音乐时界面卡顿，实现流畅交互
数据持久化：本地文件存储用户数据，单机多用户程序，针对家庭/聚会/教室等场景，多账号同一设备独立数据存储
异常处理：完善的错误捕获和用户提示
